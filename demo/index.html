STICKY MENU JS

The simplest sticky menu ever

What is this ?

it's a simple plugin that create your head-menu looks Sticky and amazing. it's lightweight and the animation are based on CSS3 without any additional CSS to add.

Javascript

$(".offset-menu").stickyMenu({ scrollAnchor: ".paragraph" });

Options

right now the stickymenu has several options to work with.

Name Default value Description
scroll true Trigger the menu animation on scroll
toggle false Trigger the menanimation on toggle or click
reverse false the hide/show menu animation are reverse
anchor 100 Trigger the menu animation after window position reached the value.
you can use selector as the anchor ( it will get the top offset position of the
selected anchor )
ex : ".paragraph"
speed 300 (in ms) menu animation speed
toggleBtn ".pull" contain selector to apply toggle

How to Customize

The core of the main animations are the CSS 3. you can customize your animation by override the selector class/id.

SCSS

.offset-menu-2{
ul{
width: 25%;
}

&.active{
ul{
width: 100%;
transition-delay: 500ms;
}
}
}
